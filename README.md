# Satria Nur Alfata Panca - Data Science Portfolio

A modern, responsive personal portfolio website for a Data Science student.

## Features

- Clean, minimalist design with gradient backgrounds
- Hero section with floating skill badges
- Responsive navigation with mobile hamburger menu
- Skills section with progress bars
- Portfolio section with filtering and modal details
- Education & Experience timeline
- Certifications & Achievements grid
- Contact form and social links
- Smooth animations and transitions
- Fully responsive for all devices

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Font Awesome for icons

## Setup Instructions

1. Clone or download the project files
2. Open `index.html` in your web browser

## Required Images

Place the following images in the `assets/images/` folder:

- `profile.jpg` - Circular profile photo (recommended: 400x400px)
- `project1.jpg` - Customer Churn Prediction project thumbnail
- `project2.jpg` - Image Classification project thumbnail
- `project3.jpg` - Sales Dashboard project thumbnail
- `project4.jpg` - Vulnerability Scanner project thumbnail
- `project5.jpg` - Sentiment Analysis project thumbnail
- `project6.jpg` - Recommendation System project thumbnail

## Customization

- Update personal information in `index.html`
- Modify colors in `assets/css/style.css`
- Add or remove projects in the portfolio section
- Update contact information and social links

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
